# 01 — 게임 개요

## 컨셉

**부서진 왕국의 연대기 (Chronicle of the Shattered Realm)** — 다크 판타지 웹 RPG.
레퍼런스: 탐험은 「어나더 에덴」(2.5D 횡스크롤 + 갈림길 레인), 마을은 「Farland Tactics」(시설 선택형), 전투는 클래식 JRPG 턴제 커맨드, 성장은 Might & Magic류 스킬 숙련 시스템.

- 해상도: 1280×720 고정 캔버스, CSS로 뷰포트에 맞춰 스케일 (`globals.css`의 `.game-root canvas`)
- 파티: 고정 4인 — **에런**(균형 리더) / **리엔**(마력) / **카시우스**(완력) / **미라**(속도)
- 세계: 변경 마을 **리븐홀드** ↔ 필드 **황혼의 숲** (단일 맵, `WORLD_W = 3400`)

## 4개 모드

모든 모드는 `src/game/scenes/`의 씬 빌더 함수이며, 좌상단 모드 배지(`setModeBadge`)로 구분 표시된다.

### 1. 탐험 모드 (`explore.ts`)
- 2.5D 횡스크롤 + **3개 레인**(깊이). 레인 이동(↑↓)은 **갈림길 구간에서만** 가능.
- 일반 몬스터: 이동 중 확률 인카운터. 정예/보스/에픽: 맵에 **심볼**로 배치.
- 리더 뒤로 파티원 3명이 궤적(trail)을 따라 후행.
- 같은 레인의 오브젝트만 상호작용 가능(`[Z] 프롬프트`).

### 2. 마을 모드 (`town.ts`)
- 시설 5곳: 무기점 / 방어구점 / 도구점 / 여관(30G 전원 완전 회복) / 모험가 길드(퀘스트 안내 + **멤버별 전직**).
- 장비 구매 시 장착할 멤버를 선택(`pickMember` 오버레이).

### 3. 이벤트 모드 (`event.ts`)
- 초상화 + 타자기 텍스트 + 선택지 분기. 스토리(서장/종장/외전)와 보스 조우 연출에 사용.

### 4. 전투 모드 (`battle.ts`)
- 이동 없는 순수 턴제 커맨드. 아군 4 + 적 1~4가 **속도 내림차순 통합 턴 큐**로 행동.
- 커맨드: 공격 / 스킬 / 방어 / 아이템 / 도망(일반 몹 전투만).

## 진행 플로우 (크리티컬 패스)

```
타이틀 → 서장 이벤트 → 마을(리븐홀드)
  → 탐험: 황혼의 숲 (레벨링, 상자 2개, 정예 오크)
  → Lv3: 길드에서 1차 전직 (멤버별, 4택1)
  → Lv6: 2차 전직 (최종 8클래스, 일부는 빛/어둠 선택)
  → x=3140 가운데 레인: [보스] 숲의 군주 그림바크 (사전 이벤트 → 전투)
  → 승리 시 종장 이벤트 → 아래 레인 x=2620에 [에픽] 고대 정령 아스테리온 등장
  → 에픽 격파 → 외전(클리어) 이벤트 → 마을
```

상태 플래그: `G.explore.defeated.{orc,lord,ancient}`, `G.flags.{intro,ending}` (`state.ts`).

## 조작

| 입력 | 동작 |
|---|---|
| ← → (A/D) | 탐험 이동 |
| ↑ ↓ | 레인 이동 (**갈림길 구간에서만**, 벗어나면 토스트 안내) |
| Z / 스페이스 | 조사 / 대화 진행 |
| Enter / 스페이스 | 타이틀에서 게임 시작 |
| 마우스/터치 | 모든 UI 버튼, 탐험 좌하단 터치 패드 ◀▶▲▼✦ |
